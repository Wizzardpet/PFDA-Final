### Description
Text interactive game set in a candyland themed zombie apocalypse. Player can search for supplies, deal weaponry, or train a zombie hoard.
Game utilizes digital art and code

## Features
-Character interaction
  -player can communicate/talk with characters they meet via text/chat boxes that take from a players chosen response to then give the respective consequence/action
-Choice based navigation
  -player actions/choices influences where they end up in the game
-inventory management (possibly??)
  -character t=can store items they choose to collect at points in the game

## Challenges
-storing items/allowing them to be used
-utilizing/making art and sprites correctly
-making a cohesive choice based game
