# Brains and Brownies (name TBD)

## Repository
https://github.com/Wizzardpet/PFDA-Final.git

### Description
Interactive game set in a candyland themed zombie apocalypse. Player can fight zombies, collect power ups, and grab weapons(?).
Game utilizes digital art and code

## Features
-Character interaction
  -player can communicate/talk with characters they meet 
-Increasing difficulty
  -levels or enemies get worse as player advances
-inventory management (possibly??)
  -character can store items they choose to collect at points in the game

## Challenges
-storing items/allowing them to be used
-dealing damage to enemy/taking damage
-making a cohesive side scroller game

# Outcomes
Ideal Outcome:
  -Game is fully functional, doesn't suck, and has all listed components

Minimal Viable Outcome:
  -Game is somewhat functional and player can at least run/swing through level
